# Constraints

## Deployment
- Target: VPS (self-hosted, Docker)
- Region: nieokreślony

## Scale
- Launch: do 100 użytkowników
- 6 months: nieznane — aplikacja nie jest projektowana pod duży ruch

## Budget
- Infrastructure: free tier gdzie możliwe (VPS własny)
- Tooling: brak limitu na narzędzia deweloperskie

## Compliance & Security
- GDPR: podstawowa zgodność (minimalne zbieranie danych, haszowane hasła)
- Brak HIPAA, SOC2, ani innych wymogów branżowych

## Tech Stack Preferences
- Language: TypeScript (frontend i backend)
- Framework: Next.js (fullstack — App Router)
- Database: PostgreSQL
- ORM: Prisma
- Testing: Vitest
- Containerization: Docker
- API style: do ustalenia w fazie Design (REST przez Next.js API routes lub Server Actions)
